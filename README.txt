JSL Omega Coin Subdomain Entry

Deploy this folder to coin.jsl-ian.com.

index.html uses the same JSL-ian homepage/browser structure:
- Logo / trademark / slogan / copyright remain on the page.
- Search box still opens URLs or sends keywords to discovery.jsl-ian.com.
- Original external links are replaced by only two links:
  Coin Sign In
  Coin Sign Up

coin.html is the coin account page:
- Sign up / sign in
- First account becomes Genesis Owner in local test mode
- User account can create/open Omega Wallet

This is an alpha/front-end test package. It is not a production cryptocurrency backend.
